# Task 4.2: layout 프로퍼티 적용

## 단계
**단계 4: 인터랙션 및 애니메이션 적용 (Motion UI)**

## 설명
Framer Motion의 layout 프로퍼티를 적용하여 항목 이동 및 필터링 시 부드러운 위치 재배치를 구현합니다.

## 체크리스트
- [ ] motion 컴포넌트에 layout 프로퍼티 추가
- [ ] 필터 전환 시 재배치 애니메이션 확인
- [ ] 항목 순서 변경 시 애니메이션 확인
- [ ] 레이아웃 전환 스무딩 조정

## 주요 기능
- 필터 변경 시 항목들이 부드럽게 재배치
- 항목 추가/삭제 시 다른 항목들이 자연스럽게 이동
- 레이아웃 변화의 자동 애니메이션

## 참고사항
- layout 프로퍼티는 자동으로 레이아웃 변화를 감지하고 애니메이션합니다
- LayoutGroup을 사용하면 여러 컴포넌트 간 레이아웃 동기화 가능
