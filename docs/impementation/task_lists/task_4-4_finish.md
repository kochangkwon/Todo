# Task 4.4: 프로그레스 바 애니메이션

## 단계
**단계 4: 인터랙션 및 애니메이션 적용 (Motion UI)**

## 설명
성취도 달성에 따른 프로그레스 바의 부드러운 게이지 이동을 구현합니다.

## 체크리스트
- [ ] 프로그레스 바 값 변경 시 애니메이션 적용
- [ ] 스무스한 전환 효과 설정
- [ ] 달성률 숫자 카운팅 애니메이션
- [ ] 색상 전환 효과 (선택사항)

## 주요 기능
- 완료율 변경 시 프로그레스 바가 부드럽게 이동
- 숫자도 카운팅되는 효과
- 100% 달성 시 특별한 효과 (선택사항)

## 참고사항
- motion.div로 프로그레스 바 래핑
- animate 프로퍼티로 값 변화 애니메이션
- spring 트랜지션으로 자연스러운 움직임
