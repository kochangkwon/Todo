# Task 4.1: AnimatePresence를 이용한 애니메이션

## 단계
**단계 4: 인터랙션 및 애니메이션 적용 (Motion UI)**

## 설명
Framer Motion의 AnimatePresence를 사용하여 리스트 항목 추가/삭제 시 슬라이드 애니메이션을 구현합니다.

## 체크리스트
- [ ] Framer Motion 설치
- [ ] AnimatePresence 컴포넌트 적용
- [ ] 항목 추가 애니메이션 구현
- [ ] 항목 삭제 애니메이션 구현
- [ ] 슬라이드 효과 설정

## 명령어
```bash
npm install framer-motion
```

## 주요 기능
- 새 항목 추가 시 부드럽게 나타남
- 항목 삭제 시 부드럽게 사라짐
- 슬라이드 효과로 자연스러운 전환

## 참고사항
- AnimatePresence는 컴포넌트가 DOM에서 제거될 때 애니메이션을 가능하게 합니다
- 각 항목에 고유한 key 값이 필요합니다
