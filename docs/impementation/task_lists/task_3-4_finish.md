# Task 3.4: TodoList 컴포넌트 구현

## 단계
**단계 3: 핵심 UI 컴포넌트 개발 (Static UI)**

## 설명
필터링 기능(전체/진행중/완료)이 포함된 탭 UI를 구성합니다.

## 체크리스트
- [ ] TodoList 컴포넌트 파일 생성
- [ ] 탭 UI 구현 (전체/진행중/완료)
- [ ] 필터링 로직 구현
- [ ] TodoItem 리스트 렌더링
- [ ] 빈 상태 처리

## 주요 기능
- 탭으로 필터 전환 (All/Active/Completed)
- 선택된 필터에 따라 할 일 목록 표시
- TodoItem 컴포넌트를 반복 렌더링

## 참고사항
- 필터 상태는 useState로 관리
- 필터링된 배열을 생성하여 렌더링
- 각 필터별 개수도 표시하면 좋습니다
