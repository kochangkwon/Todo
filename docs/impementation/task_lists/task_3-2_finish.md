# Task 3.2: TodoStats 컴포넌트 구현

## 단계
**단계 3: 핵심 UI 컴포넌트 개발 (Static UI)**

## 설명
상단 달성률 프로그레스 바 및 카운터 위젯을 구현합니다.

## 체크리스트
- [ ] TodoStats 컴포넌트 파일 생성
- [ ] 전체 할 일 개수 표시
- [ ] 완료된 할 일 개수 표시
- [ ] 달성률 계산 로직 구현
- [ ] 프로그레스 바 UI 구현
- [ ] 통계 카운터 UI 구현

## 주요 기능
- 전체 할 일 / 완료 할 일 개수 표시
- 달성률(%) 계산 및 표시
- 프로그레스 바로 시각적 표현

## 참고사항
- shadcn/ui의 Progress 컴포넌트 활용
- 달성률 = (완료 개수 / 전체 개수) * 100
