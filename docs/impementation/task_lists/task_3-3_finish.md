# Task 3.3: TodoItem 컴포넌트 구현

## 단계
**단계 3: 핵심 UI 컴포넌트 개발 (Static UI)**

## 설명
개별 할 일 항목의 레이아웃(체크박스, 텍스트, 삭제 버튼)을 구현합니다.

## 체크리스트
- [ ] TodoItem 컴포넌트 파일 생성
- [ ] 체크박스 UI 구현
- [ ] 할 일 내용 텍스트 표시
- [ ] 우선순위 뱃지 표시
- [ ] 삭제 버튼 UI 구현
- [ ] 완료 시 취소선 스타일 적용

## 주요 기능
- 체크박스 클릭으로 완료 상태 토글
- 할 일 내용 표시
- 우선순위 시각적 표시
- 삭제 버튼으로 항목 제거
