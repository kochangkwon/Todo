# Task 5.2: 데이터 백업 기능

## 단계
**단계 5: 고도화 및 UX 디테일 (Advanced UX)**

## 설명
현재 할 일 목록을 JSON 파일로 내보내기 및 가져오기 기능을 구현합니다.

## 체크리스트
- [ ] 내보내기(Export) 기능 구현
- [ ] 가져오기(Import) 기능 구현
- [ ] JSON 파일 다운로드 로직
- [ ] 파일 업로드 및 파싱 로직
- [ ] 데이터 유효성 검증

## 주요 기능
- Export: 현재 할 일 목록을 JSON 파일로 다운로드
- Import: JSON 파일을 업로드하여 할 일 목록 복원
- 데이터 백업 및 복구

## 참고사항
- Blob과 URL.createObjectURL을 사용하여 파일 다운로드
- FileReader를 사용하여 파일 읽기
- Zod로 가져온 데이터 검증
