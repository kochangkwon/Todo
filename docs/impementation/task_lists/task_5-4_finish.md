# Task 5.4: 다크 모드 최적화 및 접근성 체크

## 단계
**단계 5: 고도화 및 UX 디테일 (Advanced UX)**

## 설명
다크 모드/라이트 모드 테마를 최적화하고 접근성(A11y)을 체크합니다.

## 체크리스트
- [ ] next-themes 설치 및 설정
- [ ] 다크 모드 색상 팔레트 정의
- [ ] 테마 토글 버튼 구현
- [ ] ARIA 레이블 추가
- [ ] 키보드 네비게이션 확인
- [ ] 색상 대비 확인
- [ ] 스크린 리더 테스트

## 명령어
```bash
npm install next-themes
```

## 주요 기능
- 라이트/다크 모드 전환
- 시스템 설정 감지
- 접근성 준수

## 참고사항
- WCAG 2.1 AA 기준 준수
- 색상 대비율 4.5:1 이상
- 키보드로 모든 기능 접근 가능해야 함
- alt 텍스트, aria-label 적절히 사용
