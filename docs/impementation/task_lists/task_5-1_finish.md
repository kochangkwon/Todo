# Task 5.1: Empty State 처리

## 단계
**단계 5: 고도화 및 UX 디테일 (Advanced UX)**

## 설명
할 일이 없을 때 보여줄 일러스트 및 Floating 애니메이션을 구현합니다.

## 체크리스트
- [ ] Empty State 컴포넌트 생성
- [ ] 일러스트 또는 아이콘 추가
- [ ] 안내 메시지 작성
- [ ] Floating 애니메이션 구현
- [ ] 각 필터별 Empty State 메시지 차별화

## 주요 기능
- 할 일이 없을 때 친근한 메시지 표시
- 아이콘이나 일러스트가 떠다니는 효과
- 사용자가 할 일을 추가하도록 유도

## 참고사항
- Framer Motion으로 y축 움직임 구현
- 각 필터(전체/진행중/완료)별 다른 메시지
- lucide-react 아이콘 라이브러리 활용 가능
