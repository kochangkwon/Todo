# Task 4.3: 마이크로 인터랙션 구현

## 단계
**단계 4: 인터랙션 및 애니메이션 적용 (Motion UI)**

## 설명
사용자 인터랙션에 대한 섬세한 피드백 애니메이션을 구현합니다.

## 체크리스트
- [ ] 체크박스 클릭 시 'V' 표시 드로잉 애니메이션
- [ ] 버튼 호버 시 Scale 효과
- [ ] 버튼 클릭 시 Scale 피드백
- [ ] 우선순위 뱃지 애니메이션

## 주요 기능
- 체크박스: 체크 시 V 마크가 그려지는 효과
- 버튼: 호버 시 살짝 확대, 클릭 시 축소 후 복원
- 부드러운 전환으로 사용자 경험 향상

## 참고사항
- whileHover, whileTap 프로퍼티 활용
- SVG path 애니메이션으로 체크마크 드로잉 효과
- transition 설정으로 애니메이션 속도 조절
