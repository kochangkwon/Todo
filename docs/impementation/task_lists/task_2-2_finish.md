# Task 2.2: Zod를 이용한 입력값 검증 스키마 작성

## 단계
**단계 2: 데이터 스키마 및 상태 관리 (Business Logic)**

## 설명
Zod 라이브러리를 사용하여 할 일 입력값에 대한 검증 스키마를 작성합니다.

## 체크리스트
- [ ] Zod 라이브러리 설치
- [ ] Todo 생성 스키마 정의
- [ ] 입력값 검증 규칙 설정
- [ ] 에러 메시지 정의

## 명령어
```bash
npm install zod
```

## 참고사항
- 내용(content)은 최소 1자 이상이어야 합니다
- 우선순위는 정해진 값만 허용해야 합니다
- Zod의 타입 추론 기능을 활용하세요
